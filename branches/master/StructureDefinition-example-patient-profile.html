Resource Profile: Example Patient Profile

Official URL: http://example.com/fhir/example/StructureDefinition/example-patient-profile Version: 1.0.0
Active as of 2022-07-14 Computable Name: PEMHbundle

O Bundle da mensagem deve ser do tipo ‘message’, deve ter como primeira entrada o recurso MessageHeader, e como segunda o recurso.

Usage:

  • This Resource Profile is not used by any profiles in this Implementation Guide
